# Franka Panda MuJoCo Mesh Assets

These STL meshes are copied from the MuJoCo template assets in:

```text
dynamic-cloth-folding
https://github.com/hietalajulius/dynamic-cloth-folding
```

The upstream project is MIT licensed. A copy of the upstream license is kept in
`LICENSE.dynamic-cloth-folding`.

The fold-cloth Panda demo uses the link, hand, and 3D-printed finger meshes from
that cloth-folding environment, while keeping the control and dataset writer
local to this repository.
